# 数列と級数

対応する Lean ファイル: `StudyLean4/Algebra/Sequences.lean`

## 内容

### 数学 B — 等差数列・等比数列
| 定理名 | 内容 |
|--------|------|
| `arith_diff` | 等差数列の隣接差: $a_{n+1}-a_n = d$ |
| `geom_ratio` | 等比数列の公比: $a_{n+1}/a_n = r$ |

### 数学 B — 和の公式
| 定理名 | 内容 |
|--------|------|
| `my_gauss` | ガウスの公式: $\sum_{k=0}^{n} k = \dfrac{n(n+1)}{2}$ |
| `sum_sq` | 平方和: $\sum_{k=0}^{n} k^2 = \dfrac{n(n+1)(2n+1)}{6}$ |
| `sum_cubes` | 立方和: $\sum_{k=0}^{n} k^3 = \left(\dfrac{n(n+1)}{2}\right)^2$ |
| `sum_arith` | 等差数列の和 |
| `sum_geom` | 等比数列の和: $(r-1)\sum r^k = r^n-1$ |
| `sum_geom_div` | 等比数列の閉じた形 (r≠1) |
